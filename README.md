# Bank-Loan-Analysis

# Project Overview
This project focuses on analyzing bank loan data to uncover meaningful business insights related to loan applications, funded amounts, repayments, customer profiles, and loan performance.
The main objective of this project is to help financial institutions understand lending trends, identify risk factors, and monitor overall loan performance through interactive dashboards and SQL analysis

# Tools & Technologies Used
- SQL – Data extraction, joins, aggregations, KPI calculations, and analytical queries
- Excel – Data cleaning, preprocessing, pivot tables, and dashboard creation
- Power BI – Interactive dashboard development and business insights visualization
- Tableau – Advanced data visualization and storytelling dashboards

# Key Analysis Performed
- Total Loan Applications Analysis
- Funded Amount & Received Amount Analysis
- Good Loan vs Bad Loan Analysis
- Interest Rate & Debt-to-Income (DTI) Analysis
- Loan Status Tracking
- Regional & Monthly Loan Trends
- Customer Financial Behavior Insights

# Project Outcome
This project enhanced my practical knowledge of data analytics, dashboard development, SQL querying, and business intelligence reporting while converting raw financial data into actionable insights.
